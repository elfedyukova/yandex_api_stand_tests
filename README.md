# Тесты на проверку параметра name при создании набора в Яндекс.Прилавок с помощью API Яндекс.Прилавок. 

- Для запуска тестов должны быть установлены пакеты pytest и requests 

- Запуск всех тестов выполняется командой pytest



• Установка: Шаги по клонированию репозитория, созданию виртуального окружения и установке зависимостей.
• Использование: Пример кода, показывающий, как использовать библиотеку requests.
• Автотесты: Инструкции по запуску автотестов с использованием pytest.
